overflow-auto: This property will add scrollbars only when the content overflows the container. If the content fits within the container, no scrollbars will be displayed.
overflow-scroll: This property will always display scrollbars, regardless of whether the content overflows the container or not. Even if the content fits perfectly within the container, scrollbars will still be visible.
white-space no wrap : horizontal scroll will be added if the content exceeds the width of the container. The text will not wrap to the next line, and instead, it will continue on a single line, causing a horizontal scrollbar to appear if necessary.
textoverflow : elipses will be added to the end of the text if it exceeds the width of the container. This is often used in conjunction with overflow: hidden to create a truncated text effect.
