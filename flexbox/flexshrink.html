flex shrink is the property that defines the ability for a flex item to shrink if necessary. It accepts a unitless value that serves as a proportion. It dictates how much the item will shrink relative to the rest of the flex items in the container when there isn't enough space on the line. The default value is 1, which means that all items will shrink equally when necessary. If an item has a flex-shrink value of 0, it will not shrink at all, even if there is not enough space in the container.
