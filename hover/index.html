hover means when you put your mouse on the element it will change the color and text color and when you remove your mouse from the element it will return to its original color and text color this called hover effect
and you can use it in many ways like changing the background color, text color, font size, and many more
transition is used to make the hover effect smooth and it takes how much second seconds we are giving to the transition to complete that time it will change the color and text color and when we remove the mouse from the element it will return to its original color and text color in the same time we are giving to the transition
