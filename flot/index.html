flot is a CSS property that allows elements to be taken out of the normal flow of the document and positioned to the left or right of their container. When an element is floated, it will be aligned to the left or right edge of its containing block, and other content will flow around it. This can be useful for creating layouts where you want text or other elements to wrap around an image or a block of content.
display flow-root is a CSS property that creates a new block formatting context for the element it is applied to. This means that the element will contain its floated children and prevent them from affecting the layout of other elements outside of it. When an element has display: flow-root, it will establish a new block formatting context, which allows it to contain floated elements without collapsing its height or affecting the layout of other elements on the page. This can be particularly useful when you want to create a container that holds floated elements without causing layout issues for other parts of the page.
