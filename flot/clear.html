clear is a property that specifies on which sides of an element floating elements are not allowed to float. It can take the following values:
  • left: No floating elements allowed on the left side.
  • right: No floating elements allowed on the right side.
  • both: No floating elements allowed on either the left or the right side.
  • none: Floating elements are allowed on both sides. This is the default value.
