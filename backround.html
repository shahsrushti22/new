propertyes of the backround-repeat

  1. repeat
  2. no repeat
  3. repeat-x
  4. repeat-y

repeat

repeat the image both horizontally and vertically to fill the entire element.

no repeat

no repeat the image will only be displayed once, and it will not be repeated in any direction.

repeat-x

repeat the image only horizontally, meaning it will be repeated across the width of the element but not vertically.

repeat-y

repeat the image only vertically, meaning it will be repeated down the height of the element but not horizontally.
