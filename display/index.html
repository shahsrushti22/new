display inline and inline block

display inline and inline block are similar but they have some differences. display inline does not respect the width and height properties, while display inline block does. inline elements are not affected by the top and bottom margins, while inline block elements are.
inline block elements can have padding and margins, while inline elements cannot.

display none

display none is used to hide an element from the page. visible elements take up space on the page, while display none elements do not.
i am a display none element
display none elements do not take up space on the page, while visible elements do. display none elements are not rendered on the page, while visible elements are.
