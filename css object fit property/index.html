object-fit: cover; will make the image cover the entire container while maintaining its aspect ratio. If the image's aspect ratio does not match the container's aspect ratio, it will be cropped to fit.
object-fit: contain; will make the image fit within the container while maintaining its aspect ratio. If the image's aspect ratio does not match the container's aspect ratio, it will be letterboxed (i.e., there will be empty space on either side of the image).
object-fit is a CSS property that allows you to control how an image or video is resized to fit within its container. It is particularly useful when the aspect ratio of the media does not match the aspect ratio of the container, and you want to control how the media is displayed. by deafult object-fit is set to fill which will stretch the image to fit the container without maintaining its aspect ratio. You can use other values like cover, contain, none, and scale-down to control how the image is displayed within its container.
