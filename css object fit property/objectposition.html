object-position: top right; will position the image to the top right corner of the container. You can use other values like center, bottom left, etc., to position the image as desired. object-position can be used in conjunction with object-fit to control how the image is displayed within its container.
object position is a property that allows you to control the position of an image or video within its container when using the object-fit property. It is particularly useful when the aspect ratio of the media does not match the aspect ratio of the container, and you want to control which part of the media is visible.
